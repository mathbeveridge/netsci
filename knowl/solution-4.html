Solution 1.1.4.1.

  1. A "simple digraph" is a digraph without loops and without multiple arcs. You are allowed to have both \(uv\) and \(vu\) in a digraph. There are

    \begin{equation*} n (n-1) = (n)_2 \end{equation*}

    ways to create an arc. We have \(n\) choices for the starting vertex and \(n-1\) choices for the target vertex.

  2. Here are two solutions.

    • We must decide whether each arc is in the digraph. We must make \(n(n-1)\) yes/no independent decisions, so the number of simple digraphs is

      \begin{equation*} 2^{n(n-1)}. \end{equation*}

    • There are now four choices for every vertex pair: no arc, just \((u,v)\text{,}\) just \((v,u)\) or both \((u,v), (v,u)\text{.}\) We must make this decision for every pair of vertices. So the total number of digraphs is

      \begin{equation*} 4^{n \choose 2}. \end{equation*}

    Of course, these two numbers are equal.
