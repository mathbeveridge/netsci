Try to import the airline graph and find its centrality measures. Googling the following terms will bring up useful parts of the NetworkX documentation:

networkx tutorial
networkx centrality
networkx pagerank
networkx graphml import

Or visit our moodle site for direct links. If you want to plot the network via python, here is a simple code snippet that gives you an idea of how to start.

import networkx as nx
import matplotlib.pyplot as plt

G = nx.cubical_graph()
nx.draw(G)
plt.show()
