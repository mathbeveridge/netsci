Show that the clustering coefficient is

\begin{equation*} C = \frac{3(c-2)}{4(c-1)}. \end{equation*}

Furthermore, for \(c \geq 3\text{,}\) we have \(\frac{3}{8} \leq C \leq \frac{3}{4}\text{.}\) This is independent of the network size \(n\text{,}\) so we have high clustering.
