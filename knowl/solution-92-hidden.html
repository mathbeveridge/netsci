• The probability that we consider copying a citation to paper \(i\) is \(\frac{k_i}{cn}\text{.}\) The probability that we do copy that citation is \(\gamma\text{.}\) We try this \(c\) times.

  • The expected number of random citations we make is \(c(1-\gamma)\text{.}\) A paper has probability \(\frac{1}{n}\) of being chosen each time.

Note: Assuming that \(n\) is large compared to \(c\text{,}\) the probability that \(i\) gets multiple citations is negligible, so below, we will use the approximation \(\P\)(paper \(i\) receives a citation) \(\approx \E(\#\) citations received by paper \(i\)).
