Solution 12.3.1.

  1. An edge corresponds to an unordered pair of vertices, chosen from the set \([n]\text{.}\) In other words, we must choose a subset of size 2 from the set \([n]\text{.}\) There are

    \begin{equation*} {n \choose 2} = \frac{n(n-1)}{2} \end{equation*}

    ways to do this.

  2. Let's give a recipe for creating a graph on the vertices \([n]\text{.}\) For each pair of vertices, we must decide whether or not the corresponding edge is in our graph. So we must make \({n \choose 2}\) yes/no decisions, and these decisions are made independently of one another. Therefore, the total number of graphs is

    \begin{equation*} 2^{n \choose 2}. \end{equation*}
