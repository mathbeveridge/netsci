Section 5.1 Network Cascades

The linear threshold model is a dynamic process for diffusion of innovation on a network. We are given a network and a threshold \(0 < q < 1\text{.}\) The network starts with all vertices using behavior \(B\text{.}\) We pick a seed set of innovators who adopt behavior \(A\text{.}\) We then watch the innovation spread according to the following rule. Let \(p_v\) be the fraction of neighbors of vertex \(v\) that have adopted behavior \(A\text{.}\) The vertex \(v\) will:

  • adopt behavior \(A\) if \({p \geq q}\)

  • adopt behavior \(B\) if \({p < q}\)

We cycle through all the vertices until the update process stabilizes. The resulting adoption process is called a cascade. We have a complete cascade when every vertex adopts the new behavior \(A\text{.}\)

Exercises Practice Problems

1. Network Cascades from Singleton Seed Sets.

Consider this network with linear threshold \(q=1/3\text{.}\) Start a cascade by picking any single vertex to be the initial adopter. Does it spread to the entire network? What about the other vertices? Which ones lead to complete cascades?

Solution.

For \(q=1/3\text{,}\) every singleton induces a complete cascade.

2. Network Cascades from Doubleton Seed Sets.

Now set the threshold \(q=1/2\text{.}\) Using the same graph as in problem 1, determine the cascades for each of the following sets \(S_0\) of early adopters. Which initial sets induce complete cascades?

  1. \(\displaystyle S_0 = \{1,2\}\)

  2. \(\displaystyle S_0 = \{2,3\}\)

  3. \(\displaystyle S_0 = \{1, 5\}\)

Solution.

  1. \(S_0 = \{1,2\}\text{:}\) complete cascade

  2. \(S_0 = \{2,3\}\text{:}\) complete cascade

  3. \(S_0 = \{1, 5\}\text{:}\) only vertex 7 adopts the innovation

3. Network Cascades that are Not Complete.

Continue using threshold \(q=1/2\) for the network above. Find all of the initial adopter sets of size 2 that do not lead to complete cascades. Can you describe the network obstruction that prevented the cascade from expanding across the entire network? Solution.

The 2-sets that do not spark complete cascades are: \(\{1,5\}\) and \(\{1,7\}\) and \(\{5,7\}\) and \(\{1,3\}\text{.}\) For the first three sets, the cascade cannot jump out of the triangle \(T=\{1,5,7\}\text{.}\) The vertices that border set \(T\) have more neighbors in \(\overline{A}\) than in \(A\text{,}\) so they never adopt. Starting with set \(\{1,3\}\text{,}\) there is no cascade.

4. Complete Cascades and Cohesive Sets.

Let \({G=(V,E)}\) be a network with threshold \(q \in [0,1]\text{.}\) Let \({S_0}\) be the initial set of innovators. Recall that a set \(S\) is cohesive when

\begin{equation*} p(S) = \min_{i \in S} \frac{ | N(i) \cap S| }{|N(i)|} > 1-q \end{equation*}

where \(N(i)\) is the set of neighbors of \(i\text{.}\)

  1. Suppose that the initial adopter set \({S_0}\) does not cause a complete cascade. Prove that there is a cohesive set \({S \subset V \backslash S_0}\text{.}\) Solution.

    Let \(T\) be the set of vertices that adopt behavior \(A\) during the cascade. Let \(S= \overline{T}\) be the vertices that never adopt the behavior. For every \(i \in S\text{,}\) we have

    \begin{equation*} \frac{ | N(i) \cap S| }{|N(i)|} > 1-q. \end{equation*}

    (Otherwise, \(i\) would have adopted behavior \(A\text{.}\)) But this means that

    \begin{equation*} p(S) = \min_{i \in S} \frac{ | N(i) \cap S| }{|N(i)|} > 1-q. \end{equation*}

    Therefore the set is cohesive.

  2. Suppose that there is a cohesive set \({S \subset V \backslash S_0}\text{.}\) Prove that initial adopter set \({S_0}\) does not cause a complete cascade. Solution.

    We claim that no vertex in \(S\) ever adopts behavior \(A\text{.}\) Assume for the sake of contradiction that \(i \in S\) is the first vertex in \(S\) to adopt the innovation \(A\text{.}\) But even if every vertex outside of \(S\) has adopted behavior \(A\text{,}\) we still have

    \begin{equation*} \frac{ | N(i) \cap S| }{|N(i)|} > 1-q, \end{equation*}

    and none of the vertices in \(S\) have adopted the innovation. So \(i\) does not adopt \(A\text{,}\) a contradiction. Therefore no vertex in \(S\) adopts the innovation.
