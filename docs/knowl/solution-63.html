Solution 1.

  • As \(n \rightarrow \infty\text{,}\) the expected value \(\E[X]\) is

    \begin{equation*} \lim_{n \rightarrow \infty} \frac{C}{-\alpha +2} \left( \frac{1}{n^{\alpha-2}} - \frac{1}{k_{\min}^{\alpha-2}} \right) = \frac{C}{\alpha-2} \cdot \frac{1}{k_{\min}^{\alpha-2}} \end{equation*}

    provided that \(\alpha > 2\text{.}\) So the average value is finite if and only if \(\alpha > 2\text{.}\) Finally, let's substitute in our approximation of the constant \(C = (\alpha-1)k_{\min}^{\alpha-1}\) to find that the expected value is

    \begin{equation*} \E[X] = \frac{\alpha-1}{\alpha-2} \cdot \frac{k_{\min}^{\alpha-1}}{k_{\min}^{\alpha-2}} = \frac{\alpha-1}{\alpha-2} \cdot k_{\min}. \end{equation*}

    That's pretty cool. And you can see that if your exponent \(\alpha \rightarrow 2\) then the expected value goes to infinity. Nice!

  • As \(n \rightarrow \infty\text{,}\) the \(m\)th moment \(\E[X^m]\) is

    \begin{equation*} \lim_{n \rightarrow \infty} \frac{C}{-\alpha +m+1} \left( \frac{1}{n^{\alpha-m-1}} - \frac{1}{k_{\min}^{\alpha-m-1}} \right) = \frac{C}{\alpha-m-1} \cdot \frac{1}{k_{\min}^{\alpha-m-1}} \end{equation*}

    provided that \(\alpha > m\text{.}\) So the \(m\)th moment value is finite if and only if \(\alpha > m\text{.}\) Let's substitute \(C= (\alpha-1)k_{\min}^{\alpha-1}\) to get the expression

    \begin{equation*} \E[X^m] = \frac{\alpha-1}{\alpha-m-1} \cdot \frac{k_{\min}^{\alpha-1}}{k_{\min}^{\alpha-m-1}} = \frac{\alpha-1}{\alpha-m-1} \cdot k_{\min}^{m}. \end{equation*}
