Notes about the binomial distribution:

  • See Exercise 2.2.2 for the basics about the binomial distribution.

  • See Exercise 2.2.5 to see how we can approximate the binomial distribution with a Poisson distribtuion. Namely, when \(k\) is very small, \(n\) is very large and \(p= c/n\text{,}\) we have

    \begin{equation*} p_k \approx e^{-c} \frac{c^k}{k!}. \end{equation*}
