This activity explores a simulation of "The Emergence of the Giant Component" in the classic Random Graph Model. We will use the flexible NetLogo modeling environment. The simulation is available online:

  • Go to http://www.netlogoweb.org/

  • Click on "Net Logo Web"

  • Enter "Giant Component" in the Search Models Library input box and click on the "Sample Models/Networks/Giant Component" model.

This model starts with \(n\) isolated vertices. It then adds random edges, one at a time. You can watch how the graph evolves as it moves from an empty graph (no edges) to a complete graph (all edges). The plot on the left tracks the number of connections per node (the average degree) against the fraction of vertices in the giant component.
