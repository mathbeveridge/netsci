The number of children \(Y\) produced by each individual is an independent random variable governed by (the same) probability distribution. The expected number of children produced is

\begin{equation*} \E[Y] = \sum_{k=0}^{\infty} k \P[Y=k] = \mu. \end{equation*}

The goal of this activity will be to show that \(\mu\) determines the probability of whether the family tree dies out after a finite number of generations, or if it grows infinitely large.
