We are now ready to approximate the clustering coefficient. By parts (c) and (d), we can ignore the new triangles (there are only a constant number of them). But we need the new connected triples from part (e). The clustering coefficient is

\begin{equation*} C \approx \frac { 3 n {c/2 \choose 2} } { n {c \choose 2} + c^2pn + \frac{1}{2} c^2p^2 n } = \frac { 3{c/2 \choose 2} } { {c \choose 2} + c^2p + \frac{1}{2} c^2p^2. } \end{equation*}

This is a constant (independent of \(n\)), so we have a nonzero clustering coefficient for every small world network.
