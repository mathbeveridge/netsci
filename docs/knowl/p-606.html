One critique of Price's Model is that the new paper must know about the degrees of the entire network when we choose its citations. So this is a "global" process instead of a "local" process. Consider the following process instead. Start with a network of \(n_0 > c\) papers, each pointing randomly to \(c\) others. Pick a constant \(0 < \gamma < 1\text{.}\) When paper \(n+1\) is added, it makes \(c\) citations as follows:

  • Pick a random paper \(v\) from the existing \(n\) vertices.

  • For each of the \(c\) citations of \(v\) (that is, the arcs starting at \(v\)),

    • with \(\displaystyle{\P = \gamma}\text{,}\) copy that citation

    • with \(\displaystyle{\P = 1 - \gamma}\text{,}\) cite a uniform random paper instead.

You will now show that this model also generates a power law degree distribution.

  1. Show that for each arc starting at vertex \(n+1\text{,}\) we have

    \begin{equation*} \P(\mbox{the arc points to vertex } i) = (1- \gamma) \frac{1}{n} + \gamma \frac{k_i}{cn}. \end{equation*}

    Solution.

  2. Define \(a = c \left( \frac{1}{\gamma} -1 \right)\text{.}\) Confirm that this equation is equivalent to \(\gamma = \frac{c}{c+a}\text{.}\) Solution.

  3. Plug \(\gamma = \frac{c}{c+a}\) into equation from part (a) and simplify to show that we once again get

    \begin{equation*} \P(\mbox{the arc points to vertex } i) = \frac{a+k_i}{n(c+a)}. \end{equation*}

    Solution.

  4. Finally, rewrite the power law

    \begin{equation*} \P(\deg(v) = k) \sim k^{-2-a/c} \end{equation*}

    in terms of \(\gamma\) instead of \(a\) and \(c\text{.}\) Solution.

  5. This vertex copying model leads to the same equations for the degree distribution of the resulting network. However, the actual network structure of the vertex copying model is very different from the netwok structure of Price's Model. Explain why. (Hint: start by thinking about what the vertex copying model would look like when \(\gamma\) is very very close to 1.) Solution.
