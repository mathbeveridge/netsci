The linear threshold model is a dynamic process for diffusion of innovation on a network. We are given a network and a threshold \(0 < q < 1\text{.}\) The network starts with all vertices using behavior \(B\text{.}\) We pick a seed set of innovators who adopt behavior \(A\text{.}\) We then watch the innovation spread according to the following rule. The vertex \(v\) will:

  • adopt behavior \(A\) if \({p \geq q}\)

  • adopt behavior \(B\) if \({p < q}\)

We cycle through all the vertices until the update process stabilizes. The resulting adoption process is called a cascade. We have a complete cascade when every vertex adopts the new behavior \(A\text{.}\)
