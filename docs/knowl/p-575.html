If \(B\) is an \(n \times n\) matrix and suppose that \((I-B)\) is invertible, then we know that

\begin{equation*} (I-B)^{-1} = \sum_{j=0}^{\infty} B^j = I + B + B^2 + \cdots + B^j + \cdots. \end{equation*}

Use this fact to show that

\begin{equation*} \bx = \bone + \alpha A \bone + \alpha^2 A^2 \bone + \cdots + \alpha^s A^s \bone + \cdots \end{equation*}
