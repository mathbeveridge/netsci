The expected value of a discrete random variable is The expected value of a discrete random variable is

\begin{equation*} \E[X] = \sum_{\omega \in \Omega} X(\omega) p(\omega) \end{equation*}

which is equivalent to

\begin{equation*} \E[X] = \sum_{t \in T} t \cdot p(t) = \sum_{t \in T} t \cdot \P[X=t]. \end{equation*}
