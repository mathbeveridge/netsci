Solution 1.

For each shortcut edge, there are \(c\) ring edges incident with each enpoint. So the expected number of such triples is \(\frac{cpn}{2} (c + c) = c^2 p n\)

For each pair of shortcut edges incident with the same vertex, we must choose the other endpoints. So there are \(n\) ways to choose the common vertex, and \({n-1 \choose 2}\) ways to choose the other endpoints. We multiply by \(q^2\) since this is the probability that both shortcut edges are present.
