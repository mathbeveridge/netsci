Suppose that \(G\) is an undirected graph, so that its adjacency matrix is symmetric. The undirected graph \(G\) is reducible if and only if there is a permutation matrix \(P\) such that

\begin{equation*} P^{\top} A P = \left[ \begin{array}{cc} A_{11} & 0 \\ 0 & A_{22} \end{array} \right]. \end{equation*}

Finally, we note that the matrix \(P^{\top} A P\) simply corresponds to reordering the vertices of \(G\text{.}\)
