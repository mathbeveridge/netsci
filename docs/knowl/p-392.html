The random long-range contact is chosen according to the PMF

\begin{equation*} \Pr[ v \mbox{ is } u \mbox{'s long range contact}] = C \cdot \frac{1}{d(u,v)}. \end{equation*}

Show that the normalizing constant \(C\) satisfies

\begin{equation*} C > \frac{1}{2\log n} \end{equation*}

where \(\log n= \log_2 n\text{.}\) You will need to use the following inequalities, which you saw in your calculus classes (and might have forgotten):

\begin{equation*} \sum_{i=1}^{k} \frac{1}{i} \leq 1 + \mbox{ln}(k) \leq 1 + \log_2(k). \end{equation*}
