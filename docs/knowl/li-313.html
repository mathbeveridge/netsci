Item 18.7.a.

Pick any eigenvector \(\bu_i\) and consider the recursive process

\begin{align*} \by_{i,1} &= \bu_i\\ \by_{i,t} &= \alpha A \by_{i,t-1} + \bu_i \mbox{ for } t \geq 2. \end{align*}

Use induction on \(t \geq 1\) to prove that

\begin{equation*} \by_{i,t} = \left( 1 + \alpha \kappa_i + \alpha^2 {\kappa_i^2} + \cdots + \alpha^{t-1} {\kappa_i^{t-1}} \right) \bu_i = \sum_{k=0}^{t-1} (\alpha \kappa_i)^k \bu_i. \end{equation*}
