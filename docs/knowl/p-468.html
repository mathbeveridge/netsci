Let \({G=(V,E)}\) be a network with threshold \(q \in [0,1]\text{.}\) Let \({S_0}\) be the initial set of innovators. Recall that a set \(S\) is cohesive when

\begin{equation*} p(S) = \min_{i \in S} \frac{ | N(i) \cap S| }{|N(i)|} > 1-q \end{equation*}

where \(N(i)\) is the set of neighbors of \(i\text{.}\)

  1. Suppose that the initial adopter set \({S_0}\) does not cause a complete cascade. Prove that there is a cohesive set \({S \subset V \backslash S_0}\text{.}\)

  2. Suppose that there is a cohesive set \({S \subset V \backslash S_0}\text{.}\) Prove that initial adopter set \({S_0}\) does not cause a complete cascade.
