Start by expanding the Model Info section at the bottom. Read the sections first sections: WHAT IS IT?, HOW IT WORKS, HOW TO USE IT, THINGS TO TRY.

  1. Run the model for \(n=80\) (the default size). Watch the evolution of the whole graph, as well as the evolution of the largest component itself. Watch the giant appear. Reset and repeat a few times. Describe what you see. Look at the graph that tracks the average degree versus giant component size. Increase or slow the speed of the simulation using the slider bar.

  2. Do the same for smaller networks such as \(n=20\) and \(n=40\) and for larger networks such as \(n=160\) and \(n=320\text{.}\) Compare and contrast these evolutions. For larger networks, you should uncheck the green "layout?" checkbox. This will allow the process to run much faster. You can still keep track of the giant by using the color of the stick-figure vertices.

  3. Now let's really focus in on the emergence of the giant component. This is window in which we transition from "no giant" to "yes giant." Use the "Go" button and the "Go Once" button to Stop/Start the simulation as the giant emerges.
