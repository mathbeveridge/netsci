We need \(\displaystyle{\zeta(x) = \sum_{k=1}^{\infty} k^{-x}}\) to converge so that the constant \(C\) is finite. From calculus, we recall that the series

\begin{equation*} \sum_{n=1}^{\infty} n^{-r} = \sum_{n=1}^{\infty} \frac{1}{n^r} \end{equation*}

converges if and only if \(r > 1\text{.}\) Translating this to our power law, we set \(k=n\) and \(\alpha=r\text{.}\) We must have \(\alpha > 1\) in order for the constant \(C\) to be finite.
