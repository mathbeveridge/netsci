Check that the row vector \(\bv\) satisfies

\begin{equation*} \bv \,P = \bv. \end{equation*}

We have a name for this equation: say that "\(\bv\) is a left eigenvector of the transition matrix \(P\) for eigenvalue \(\lambda=1\text{.}\)" (In other words, the eigenvectors that you are used to are actually right eigenvectors.)
