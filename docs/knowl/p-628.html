In order to prove this result, we will divide our journey from \(s\) to \(t\) into phases:

  • phase \(j\) = ring distance \(d\) to target vertex satisfies \(2^j \leq d < 2^{j+1}\text{.}\)

Note that these phases are indexed in reverse. That is, when we are far from the target \(t\text{,}\) our phase index is larger. In other words, we "count down" the phases. For example, the in the following figure, we start in Phase 3 (at source vertex \(s\)) and end in Phase 0 (at target vertex \(t\)).

Let \(X\) denote the total number of steps in our journey. Let \(X_j\) denote the number of steps taken during phase \(j\text{.}\) In the example above, our journey takes \(X=5\) steps. We have

\begin{equation*} X_3 = 1, X_2 = 1, X_1 = 2, X_0 = 1. \end{equation*}

The goal of this activity is to show that

\begin{equation*} \E[X] = \sum_{k=0}^{\log (n/2)} \E[X_k] = O\left( (\log n)^2 \right), \end{equation*}

meaning that the expected number of steps is at most \(K \cdot (\log n)^2\) for some constant \(K > 0\text{.}\) In other words, in a typical network of this kind, our journey takes at most \(O((\log n)^2)\) steps.
