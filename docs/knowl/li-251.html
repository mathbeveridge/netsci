Item 17.3.c.

Use part (b) and induction on \(N\) to prove that there is a unique acyclic tournament structure on \(N\) vertices (up to isomorphism). Or equivalently, show that we can label the vertices of a transitive tournament on \(N\) vertices using labels \([N] = \{ 1,2, \ldots, N \}\) such that if \(i >j\) then \((i,j)\) is an arc in the tournament.
