Exercise 8. Katz centrality as \(\alpha \rightarrow 0\).

Now let's talk about what happens to the Katz centrality as \(\alpha \rightarrow 0\text{.}\) You will show that for very small \(\alpha\text{,}\) the Katz centrality gives a vertex ranking that is equivalent to degree centrality, meaning that the order of the ranking is the same as the degree centrality ranking. This time, we will start with the following equation, which is satisfied by the limiting solution to the Katz recursion:

\begin{equation*} \bx = \alpha A \bx + \bone. \end{equation*}

  1. Explain why \(A \bone < n \bone\text{.}\)

  2. If \(B\) is an \(n \times n\) matrix and suppose that \((I-B)\) is invertible, then we know that

    \begin{equation*} (I-B)^{-1} = \sum_{j=0}^{\infty} B^j = I + B + B^2 + \cdots + B^j + \cdots. \end{equation*}

    Use this fact to show that

    \begin{equation*} \bx = \bone + \alpha A \bone + \alpha^2 A^2 \bone + \cdots + \alpha^s A^s \bone + \cdots \end{equation*}

  3. Next, let \(\alpha = 1/m\) where \(m > 0\) is a very large number (to be determined later). Use parts (a) and (b) to show that

    \begin{equation*} \bx = \bone + \frac{1}{m} A \bone + \bb \end{equation*}

    where \(\bb\) is a vector satisfying

    \begin{equation*} | \bb | < \frac{n}{m (m-1)} \bone. \end{equation*}

  4. Finally, take \(m = n^2\) and look at \(\bx - \bone\text{.}\) Explain why ordering of the vertices by the entries in \(\bx - \bone\) gives the same ranking as degree centrality (as long as \(n \geq 2\)). A similar argument holds for any \(m > n^2\text{,}\) so as \(\alpha \rightarrow 0\text{,}\) the ranking is the same as the degree centrality ranking.
