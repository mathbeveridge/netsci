Exercise 3. Clustering Coefficient of a Small World.

We turn the Ring World into a Small World by adding some random edges to the graph. We call these edges "shortcuts" because they traverse across the middle of the ring.

For each existing edge in the Ring World, we add a random edge with constant probability \(p\text{.}\) So we add a total of

\begin{equation*} p\frac{cn}{2} = \frac{1}{2} cpn \end{equation*}

random edges to the graph.

  1. Define

    \begin{equation*} q = \frac{\frac{cpn}{2}}{{n \choose 2}} \approx \frac{cp}{n}. \end{equation*}

    For any potential shortcut edge \(e\text{,}\) explain why

    \begin{equation*} \P[ \mbox{edge } e \mbox{ is added to the graph}] \leq q. \end{equation*}

  2. For a fixed vertex \(v_i\text{,}\) explain why

    \begin{equation*} \E[ \mbox{# shortcut edges incident with } v_i] = cp. \end{equation*}

  3. Next, we show that we have only added a constant number of new triangles. Show that

    \begin{equation*} \E[\mbox{# triangles using one shortcut edge} ] \leq n{ c \choose 2} q \approx pc^3/2. \end{equation*}

    So we are adding a constant number of triangles. For large \(n\text{,}\) these new triangles are negligible compared to the \(n { c/2 \choose 2}\) triangles already present in the ring.

  4. Triangles using 2 or 3 shortcut edges are also unlikely. Explain why

    \begin{equation*} \E[\mbox{# triangles using two shortcut edges} ]= n \frac{c}{2} (n-2) q^2 \approx \frac{c^3p^2}{2}. \end{equation*}

    and

    \begin{equation*} \E[\mbox{# triangles using three shortcut edges} ] = {n \choose 3} q^3 \approx \frac{c^3p^3}{6}. \end{equation*}

    So we only add a constant number of triangles of any kind.

  5. Turning to connected triples, show that

    \begin{equation*} \E[ \mbox{# connected triples using one shortcut edge} ] = \frac{cpn}{2} (c + c) = c^2 p n. \end{equation*}

    and that

    \begin{equation*} \E[ \mbox{# connected triples using two shortcut edges} ] = n {n-1 \choose 2} q^2 \approx \frac{c^2 p^2}{2} n. \end{equation*}

  6. We are now ready to approximate the clustering coefficient. By parts (c) and (d), we can ignore the new triangles (there are only a constant number of them). But we need the new connected triples from part (e). The clustering coefficient is

    \begin{equation*} C \approx \frac { 3 n {c/2 \choose 2} } { n {c \choose 2} + c^2pn + \frac{1}{2} c^2p^2 n } = \frac { 3{c/2 \choose 2} } { {c \choose 2} + c^2p + \frac{1}{2} c^2p^2. } \end{equation*}

    This is a constant (independent of \(n\)), so we have a nonzero clustering coefficient for every small world network.

    Use the approximation \({a \choose 2} = a^2/2\) and set \(p=1\) to obtain the rough lower bound

    \begin{equation*} C \gtrapprox \frac{3}{16}. \end{equation*}
