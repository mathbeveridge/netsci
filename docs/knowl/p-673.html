Here is very useful fact from probability. Let \(X\) be any random variable taking on positive integer values. We can write the expected value of \(X\) as

\begin{equation*} \E[X] = \sum_{k=1}^{\infty} \Pr[X \geq k]. \end{equation*}

Indeed, when \(X\) only takes on positive integer values, we have

\begin{equation*} \sum_{k=1}^{\infty} \Pr[X \geq k] = \sum_{k=1}^{\infty} k \Pr[X = k] \end{equation*}

and the right hand side is the definition of \(\E[X]\text{.}\)
