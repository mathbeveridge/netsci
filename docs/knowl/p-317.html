The random graph \(G(n,p)\) is a probability space where \(\Omega\) is the set of all possible graphs on \([n]\text{.}\) Note that the vertices are labeled, so the same graph structure appears once for each possible labeling. For every \(G \in \Omega\text{,}\) with \(k\) edges, we have

\begin{equation*} P(G) = p^k(1-p)^{{n \choose 2} -k}. \end{equation*}
