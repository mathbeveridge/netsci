We have \(E[X_0] = 1 = \mu^0.\) Assuming that \(\E[X_n] = \mu^n\text{,}\) we have

\begin{align*} \E[X_{n+1}] &= \sum_{k=0}^{\infty} \E[X_{n+1} \mid X_{n} = k] \P[X_{n} = k] = \sum_{k=0}^{\infty} k \mu \, \P[X_{n} = k]\\ &= \mu \, \sum_{k=0}^{\infty} k \, \P[X_{n} = k] = \mu \, \E[X_n] = \mu \cdot \mu^n = \mu^{n+1}. \end{align*}
