Let \(G\) be a digraph. (\(\Rightarrow\)) Suppose that its adjacency matrix is reducible, so that

\begin{equation*} P^{\top} A P = \left[ \begin{array}{cc} A_{11} & A_{12} \\ 0 & A_{22} \end{array} \right]. \end{equation*}

Then there is is no path from any vertex in the second set to any vertex in the first set. So \(G\) is not strongly connected.
