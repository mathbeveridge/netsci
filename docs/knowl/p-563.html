We have partitioned the journey into phases. Each step occurs in exactly one phase: when we are at a vertex that is distance \(d\) from the target, we are in phase \(\lfloor \log(d) \rfloor\text{.}\) So to get the total number of steps, we just add the number of steps taken in each phase. Here are a couple of notes:

  • The indexing of the phases "counts down": the first phase is \(j= \lfloor \log(n/2) \rfloor \) and the last phase is \(j=0\text{.}\)

  • It is possible to have \(X_k=0\text{.}\) this occurs when we take a long range edge that jumps over the interval \(2^k \leq d < 2^{k+1}\text{.}\)
