We have

\begin{equation*} \P[ X_t \geq 1 \mbox{ for all } t \geq 0] = \alpha >0. \end{equation*}

But then

\begin{equation*} \E[Z_n] = \sum_{t=0}^n \E[X_t] \geq \sum_{t=0}^n \P[X_t \geq 1] = \sum_{t=0}^n \alpha = \alpha (n+1) \end{equation*}

and so

\begin{equation*} \E[Z] = \lim_{n \rightarrow \infty} \E[Z] = \lim_{n \rightarrow \infty} \alpha (n+1) = \infty. \end{equation*}
