Exercises Practice Problems

1. Adjacency Matrix of a Graph.

Find the adjacency matrix \(A\) of the following undirected graph, where \(A_{ij}=1\) if \((i,j) \in E\) (an undirected edge), and is 0 otherwise.

2. Adjacency Matrix of a Digraph.

Find the adjacency matrix \(A\) of the following directed graph, where \(A_{ij}=1\) if \((i,j)\in E\) (a directed arc), and is 0 otherwise. Note: the \(i\)th row contains all the arcs leading out of \(i\text{.}\) So construct the matrix row-by-row.

3. Reducible Matrices.

A \(n \times n\) matrix is reducible if there is an \(n \times n\) permutation matrix \(P\) such that

\begin{equation*} P^{\top} A P = \left[ \begin{array}{cc} A_{11} & A_{12} \\ 0 & A_{22} \end{array} \right] \end{equation*}

where \(A_{11}\) and \(A_{22}\) are square matrices. Conjugation by \(P\) rearranges the rows and columns. So this is equivalent to saying that we can order the vertices of \(G\) so that the adjacency matrix has the specified form.

  1. Recall that if \(G\) is an undirected graph, then \(A\) is symmetric. Explain why the adjacency matrix of an undirected graph is reducible if and only if \(G\) is not connected.

  2. Prove that the adjacency matrix of a directed graph \(G\) is reducible if and only if \(G\) is not strongly connected.

4. Spectral Decomposition of a \(2 \times 2\) Matrix.

Let

\begin{equation*} A = \begin{bmatrix} -2 & 2\\ 2 & 1 \end{bmatrix}. \end{equation*}

  1. Check that the vector \(\bv_1 = \begin{bmatrix} 1 \\ 2 \end{bmatrix}\) is an eigenvector of \(A\) for eigenvalue \(\lambda_1=2\text{.}\)

  2. Check that the vector \(\bv_2 = \begin{bmatrix} 2 \\ -1 \end{bmatrix} \) is an eigenvector of \(A\) for eigenvalue \(\lambda_2 = -3\text{.}\)

  3. Check that \(\bv_1\) and \(\bv_2\) are orthogonal vectors.

  4. Confirm that \(A = \lambda_1 \bu_1 \bu_1^{\top} + \lambda_2 \bu_2 \bu_2^{\top}\) where \(\bu_1, \bu_2\) are eigenvectors of unit length in the direction of \(\bv_1, \bv_2\) respectively.

5. Repeated Multiplications and Spectral Decomposition.

Suppose that we have a \(2 \times 2\) matrix with spectral decomposition

\begin{equation*} A = \lambda_1 \bu_1 \bu_1^{\top} + \lambda_2 \bu_2 \bu_2^{\top} \end{equation*}

where \(\bu_1, \bu_2\) are orthonormal eigenvectors for eigenvalues \(\lambda_1\) and \(\lambda_2\text{.}\) Since these vectors are orthonormal, we have

\begin{equation*} \bu_1^{\top} \bu_1=1, \bu_1^{\top} \bu_2=0, \bu_2^{\top} \bu_1=0 , \bu_2^{\top} \bu_2=1. \end{equation*}

The goal of this question is to estimate \(A^n \bx\) for large \(n\text{,}\) where \(\bx \in \R^2\) is an arbitrary vector.

  1. Find the spectral decomposition for \(A^2=AA\) by using the spectral decomposition for each \(A\text{,}\) and then multiplying out.

  2. What is the spectral decomposition of \(A^3 =A^2A\text{?}\) How about the spectral decomposition of \(A^n\text{?}\)

  3. The orthonormal vectors \(\bu_1, \bu_2\) form a basis for \(\R^2\text{.}\) So we can write any vector \(\bx \in \R^2\) as \(\bx = c_1 \bu_1 + c_2 \bu_2\text{.}\) Use your answer from part (b) to find an expression for \(A^n \bx\text{.}\)

  4. Now let's suppose that \(|\lambda_1| > |\lambda_2|\) and that \(\bx\) is such that \(c_1 \neq 0\text{.}\) Explain why \(A^n \bx \approx k \bu_1\) for some constant \(k\text{.}\)

You have now shown that \(A^n \bx\) is converges to a vector in the span of the eigenvector \(\bu_1\) corresponding to the largest eigenvalue \(\lambda_1\text{.}\)
