Section 4.6 Navigable Small World Model

The Small World Model of Strogatz and Watts contains short paths, but they are very hard to find! In this activity, we study a variation of Kleinberg's Navigable Small World Model. We start with a cycle on \(n+1\) vertices \(\{0,1,2,\ldots n\}\text{.}\) Then each vertex chooses a (one-way) "long-range contact" according to

\begin{equation*} \Pr[ u \mbox{ is } v\mbox{'s long range contact}] = C \cdot \frac{1}{d(u,v)}. \end{equation*}

where \(C\) is our normalizing constant and \(d(u,v)\) is the ring distance between nodes \(u\) and \(v\text{.}\)

We consider the following greedy routing strategy from source vertex \(s\) to target vertex \(t\text{:}\)

  • move as close to the target as possible, either by walking on the ring edge, or following the long-range edge (when this takes us closer to \(t\)).

Consider a journey from source vertex \(s\) to target vertex \(t\text{.}\) We will consider the "worst case" of starting at vertex \(0\) and trying to get to vertex \(n/2\text{.}\) You will show that the expected number of steps in this journey is

\begin{equation*} O \left( (\log n)^2 \right) \end{equation*}

where \(\log n = \log_2 n\) is the base-2 logarithm of \(n\text{.}\)

In order to prove this result, we will divide our journey from \(s\) to \(t\) into phases:

  • phase \(j\) = ring distance \(d\) to target vertex satisfies \(2^j \leq d < 2^{j+1}\text{.}\)

Note that these phases are indexed in reverse. That is, when we are far from the target \(t\text{,}\) our phase index is larger. In other words, we "count down" the phases. For example, the in the following figure, we start in Phase 3 (at source vertex \(s\)) and end in Phase 0 (at target vertex \(t\)).

Let \(X\) denote the total number of steps in our journey. Let \(X_j\) denote the number of steps taken during phase \(j\text{.}\) In the example above, our journey takes \(X=5\) steps. We have

\begin{equation*} X_3 = 1, X_2 = 1, X_1 = 2, X_0 = 1. \end{equation*}

The goal of this activity is to show that

\begin{equation*} \E[X] = \sum_{k=0}^{\log (n/2)} \E[X_k] = O\left( (\log n)^2 \right), \end{equation*}

meaning that the expected number of steps is at most \(K \cdot (\log n)^2\) for some constant \(K > 0\text{.}\) In other words, in a typical network of this kind, our journey takes at most \(O((\log n)^2)\) steps.

Our analysis combines tools from computer science (analysis of algorithms), statistics (probability) and mathematics (calculus, combinatorics).

Exercises Practice Problems

1.

To check your understanding, explain why the length of the journey from \(s\) to \(t\) is

\begin{equation*} X = \sum_{k=0}^{\lfloor \log (n/2) \rfloor} X_k \end{equation*}

In other words, be sure that you understand

  • how we have partitioned the journey into phases, and

  • why the upper limit of the sum is \(\lfloor \log (n/2) \rfloor.\)

From here forward, we will omit the floor function from our expressions, and write \(\log (n/2) \) when we actually mean \(\lfloor \log (n/2) \rfloor\text{.}\) We trust that you know that we need to use integer values.

Okay, you now understand this formula for a given network. In the rest of this activity, we will calculate the expected length of a journey on a randomly generated network. We use linearity of expectation to write the expected length of this journey as the sum of the expected lengths of the phases:

\begin{equation*} \E[X] = \E \left[ \sum_{k=0}^{\log (n/2)} X_k \right] = \sum_{k=0}^{\log (n/2)} E[X_k]. \end{equation*}

Solution.

We have partitioned the journey into phases. Each step occurs in exactly one phase: when we are at a vertex that is distance \(d\) from the target, we are in phase \(\lfloor \log(d) \rfloor\text{.}\) So to get the total number of steps, we just add the number of steps taken in each phase. Here are a couple of notes:

  • The indexing of the phases "counts down": the first phase is \(j= \lfloor \log(n/2) \rfloor \) and the last phase is \(j=0\text{.}\)

  • It is possible to have \(X_k=0\text{.}\) this occurs when we take a long range edge that jumps over the interval \(2^k \leq d < 2^{k+1}\text{.}\)

2.

The random long-range contact is chosen according to the PMF

\begin{equation*} \Pr[ v \mbox{ is } u \mbox{'s long range contact}] = C \cdot \frac{1}{d(u,v)}. \end{equation*}

Show that the normalizing constant \(C\) satisfies

\begin{equation*} C > \frac{1}{2\log n} \end{equation*}

where \(\log n= \log_2 n\text{.}\) You will need to use the following inequalities, which you saw in your calculus classes (and might have forgotten):

\begin{equation*} \sum_{i=1}^{k} \frac{1}{i} \leq 1 + \mbox{ln}(k) \leq 1 + \log_2(k). \end{equation*}

Solution.

We assume that \(n\) is even (for simplicity). We have

\begin{equation*} 1 = 2 \, C \, \sum_{i=1}^{n/2} \frac{1}{i} \leq 2 \, C ( 1 + \log(n/2)) = 2 \, C ( 1 + \log(n) - \log(2)) =2 C \log(n) \end{equation*}

and so

\begin{equation*} \frac{1}{2 \log n} \leq C \end{equation*}

3.

Consider a vertex \(v\) with ring distance \(d=d(v,t)\) satisfying \(2^j \leq d < 2^{j+1}\text{.}\)

  1. Let

    \begin{equation*} A = \{ w \mid d(w,t) \leq d/2 \} \end{equation*}

    be the set of vertices that are "at most half the distance from \(t\) as \(v\text{.}\)" Explain why

    \begin{equation*} |A|=d+1 \end{equation*}

    and why

    \begin{equation*} \frac{d}{2} \leq d(v,w) \leq \frac{3d}{2} \end{equation*}

    for all \(w \in A\text{.}\) Solution.

    There are \(d+1\) vertices in \(A= \{ t - d/2, \ldots , t + d/2 \}\text{.}\) Meanwhile, vertex \(v\) satisties \(d(v,t)=d\text{,}\) so the closest \(w \in A\) can be is \(d(v,w)=d/2\) and the farthest that it can be is \(d(v,w) = 3d/2\text{.}\)

  2. Prove that

    \begin{equation*} \displaystyle{\Pr[v \mbox{'s long-range contact is in } A] \geq \frac{1}{3 \log n}}. \end{equation*}

    Solution.

    \begin{align*} \Pr[v \mbox{'s long-range contact is in } A] &\geq (d+1) \cdot C \cdot \frac{1}{3d/2}\\ & \geq (d+1) \frac{1}{ 2 \log n} \cdot \frac{2}{3d}\\ & > \frac{1}{ 3\log n} \end{align*}

  3. Use part (b) to prove that

    \begin{equation*} \displaystyle{\Pr[\mbox{round } j \mbox{ lasts at least } i \mbox{ steps}] \leq \left( 1 - \frac{1}{ 3 \log n} \right)^{i-1}}. \end{equation*}

    Solution.

    If \(v\)'s long range contact is in \(A\text{,}\) then we end the round after that step. So the probability that we take at least \(i\) steps is

    \begin{equation*} \left( 1 - \frac{1}{3 \log n} \right)^{i-1} \end{equation*}

4.

Here is very useful fact from probability. Let \(X\) be any random variable taking on positive integer values. We can write the expected value of \(X\) as

\begin{equation*} \E[X] = \sum_{k=1}^{\infty} \Pr[X \geq k]. \end{equation*}

Indeed, when \(X\) only takes on positive integer values, we have

\begin{equation*} \sum_{k=1}^{\infty} \Pr[X \geq k] = \sum_{k=1}^{\infty} k \Pr[X = k] \end{equation*}

and the right hand side is the definition of \(\E[X]\text{.}\)

Use this new formula for \(\E[X]\) and part 3(c) to prove that

\begin{equation*} E[X_j] \leq 3 \log n. \end{equation*}

Solution.

The expected length of a round is at most

\begin{equation*} \sum_{k=1}^{\infty} \Pr[X \geq k] = \sum_{k=1}^{\infty} \left( 1 - \frac{1}{3 \log n} \right)^{i-1} = \frac{1}{1 - \left( 1 - \frac{1}{ 3 \log n}\right)} = 3 \log n \end{equation*}

5.

Use problems 1 and 4 to conclude that an average journey takes at most

\begin{equation*} O((\log n)^2) \end{equation*}

steps. Solution.

We have

\begin{align*} \sum_{k=0}^{\log (n/2)} E[X_j] &\leq \sum_{k=0}^{\log (n/2)}\left( 3 \log n \right)\\ &= (3 \log n) (\log n - \log 2 + 1) = 3 (\log n)^2. \end{align*}

So an average journey takes \(O((\log n)^2)\) steps.
