Section 9.2 Events

A subset \(A \subset \Omega\) of the sample space is called an event. The probability of event \(A\) is

\begin{equation*} \P(A) = \sum_{a \in A} p_X(a). \end{equation*}

The subset \(A\) is called an "event" because we often give a description of the elements in \(A\) using some shared property, rather than directly listing the elements.

Example 9.2.1. Probability of Events.

Suppose that we pick a random vertex from our example network.

Let \(A\) be the event that the degree of the vertex is even. Let \(B\) be the event that the degree of the vertex is at least 3.

  • List the elements of \(A\text{,}\) \(B\text{,}\) \(A \cap B\) and \(A \cup B\text{.}\)

  • Next, calculate \(\P[A]\text{,}\) \(\P[B]\text{,}\) \(\P[A \cap B]\) and \(\P[A \cup B]\text{.}\)

Solution.

\begin{equation*} A = \{ 1, 6, 7\} \qquad B = \{ 2, 3, 4, 5, 6, 7 \} \end{equation*}
\begin{equation*} A\cap B = \{ 6, 7\} \qquad A \cup B = \{ 1,2,3,4,5,6,7\} \end{equation*}
\begin{equation*} \P[A] = \frac{3}{7} \qquad \P[B] = \frac{6}{7} \qquad \P[A \cap B] = \frac{2}{7} \qquad \P[A \cup B] = 1 \end{equation*}

Example 9.2.2. Union of Events.

For any sample space \(\Omega\) and any events \(A\) and \(B\text{,}\) explain why

\begin{equation*} \P[A \cup B] = \P[A] + \P[B] - \P[A \cap B]. \end{equation*}

Solution.

This follows from the inclusion-exclusion formula

\begin{equation*} |A \cup B| = |A| + |B| - |A \cap B|. \end{equation*}

Subsection 9.2.1 Conditional Probability

Given two events \(A, B \subset \Omega\text{,}\) the conditional probability of \(A\) given \(B\) is

\begin{equation*} \P[A \mid B] = \frac{\P[A \cap B]}{\P[B]}. \end{equation*}

This is the probability of event \(A\text{,}\) given that event \(B\) is true.

Example 9.2.3. Conditional Probabillity.

Pick a random vertex from our example network. Let's use the same events from the first example in this subsection:

  • Let \(A\) be the event that the degree of the vertex is even.

  • Let \(B\) be the event that the degree of the vertex is at least 3.

Calculate \(\P[A \mid B]\) and \(\P[B \mid A]\text{.}\)

Solution.
\begin{equation*} \P[A \mid B] = \frac{2}{6} = \frac{1}{3} \qquad \mbox{and} \qquad \P[B \mid A] = \frac{2}{3} \end{equation*}

Subsection 9.2.2 Independent Events and Dependent Events

Two events \(A,B \in \Omega\) are independent when

\begin{equation*} \P[A \cap B] = \P[A] \P[B]. \end{equation*}

Otherwise, these events are dependent.

Example 9.2.4. Probability of Independent Events.

Show that when \(A\) and \(B\) are independent, we have

\begin{equation*} \P[A \mid B ] = \P[A]. \end{equation*}

In other words, assuming that event \(B\) holds has no effect on the probability that event \(A\) holds. Solution.

\begin{equation*} \P[A \mid B] = \frac{\P[A \cap B]}{\P[B]} = \frac{\P[A] \, \P[B]}{\P[B]} = \P[A]. \end{equation*}

Example 9.2.5. Example of Independent Events.

Show that the events \(A\) and \(B\) from the network example problems above are dependent. Solution.

We have

\begin{equation*} \P[A \cap B] = \frac{2}{7} \qquad \mbox{ but } \qquad \P[A] \P[B] = \frac{3}{7} \cdot \frac{6}{7} = \frac{18}{49}. \end{equation*}

Since \(\P[A \mid B] \neq \P[A]\text{,}\) these are dependent events.
