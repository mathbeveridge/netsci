Chapter 1 Graphs and Digraphs

Exercises Practice Problems

1. Handshaking Lemma.

Let \(G=(V,E)\) be an undirected graph. Prove the "handshaking lemma," which states that

\begin{equation*} \displaystyle{2|E| = \sum_{v \in V} \deg(v)}. \end{equation*}
Solution.

The right hand side (RHS) sums the degrees of all the vertices. Every edge is incident with two vertices, so each edge is counted twice in the RHS. So when we add up all of the degrees, we get twice the number of edges.

2. Digraph Handshaking Lemma.

Let \(D=(V,E)\) be a digraph. Prove that

\begin{equation*} |E| = \sum_{v \in V} \deg^-(v) = \sum_{v \in V} \deg^+(v). \end{equation*}
Solution.

This is the directed graph variant of the previous problem. Every arc has a "tail" and a "head." Adding up the out degrees counts the arc tails. Adding up the in degrees counts the arc heads. In each case, every arc is counted once.

3. Counting Graphs.

Consider the vertex set \(V=[n] = \{ 1, 2, 3, \ldots, n \}\text{.}\) We want to construct a simple, undirected graph on \(V\text{.}\)

  1. How many possible edges are there between these vertices?

  2. How many different simple graphs can we construct using these vertices? (The graphs do not have to be connected.)

Solution.

  1. An edge corresponds to an unordered pair of vertices, chosen from the set \([n]\text{.}\) In other words, we must choose a subset of size 2 from the set \([n]\text{.}\) There are

    \begin{equation*} {n \choose 2} = \frac{n(n-1)}{2} \end{equation*}

    ways to do this.

  2. Let's give a recipe for creating a graph on the vertices \([n]\text{.}\) For each pair of vertices, we must decide whether or not the corresponding edge is in our graph. So we must make \({n \choose 2}\) yes/no decisions, and these decisions are made independently of one another. Therefore, the total number of graphs is

    \begin{equation*} 2^{n \choose 2}. \end{equation*}
4. Counting Digraphs.

Answer the same questions as in Problem 3, but this time for a simple directed graph.

Solution.

  1. A "simple digraph" is a digraph without loops and without multiple arcs. You are allowed to have both \(uv\) and \(vu\) in a digraph. There are

    \begin{equation*} n (n-1) = (n)_2 \end{equation*}

    ways to create an arc. We have \(n\) choices for the starting vertex and \(n-1\) choices for the target vertex.

  2. Here are two solutions.

    • We must decide whether each arc is in the digraph. We must make \(n(n-1)\) yes/no independent decisions, so the number of simple digraphs is

      \begin{equation*} 2^{n(n-1)}. \end{equation*}

    • There are now four choices for every vertex pair: no arc, just \((u,v)\text{,}\) just \((v,u)\) or both \((u,v), (v,u)\text{.}\) We must make this decision for every pair of vertices. So the total number of digraphs is

      \begin{equation*} 4^{n \choose 2}. \end{equation*}

    Of course, these two numbers are equal.

5. Strongly Connected Components.

Find the strongly connected components of the following digraphs.

Solution.

  1. There are four strongly connected components:

    \begin{equation*} \{ 1 , 2 , 3 \}, \{ 4\}, \{ 5 \}, \{ 6 \}. \end{equation*}

  2. There are two strongly connected components:

    \begin{equation*} \{1,5,7\}, \{ 2,3,4,6,8\} \end{equation*}
6. Strongly Connected Digraph.

Prove that a digraph \(D=(V,E)\) is strongly connected if and only if for every partition \(V = A \cup B\) into disjoint, nonempty sets, there is an arc in \(D\) directed from \(A\) to \(B\) and an arc in \(D\) directed from \(B\) to \(A\text{.}\)

Solution.

\((\Rightarrow)\) Suppose that \(D\) is strongly connected. Pick any partition \(V=A \cup B\) into disjoint, nonempty sets. Choose any \(a \in A\) and \(b \in B\text{.}\) Since \(D\) is strongly connected, there is a directed path \(P\) from \(a\) to \(b\text{.}\) Follow this path until you reach the first arc \((x,y)\) whose tail \(x \in A\) and whose head \(y \in B\text{.}\) This is the desired arc. By an identical argument, there is an arc from \(B\) to \(A\) because there is a directed path from \(b\) to \(a\text{.}\)

\((\Leftarrow)\) Suppose that for every partition \(V = A \cup B\) into disjoint, nonempty sets, there is an arc in \(D\) directed from \(A\) to \(B\) and an arc in \(D\) directed from \(B\) to \(A\text{.}\) We show that for any vertices \(x,y\) there is a path from \(x\) to \(y\) and from \(y\) to \(x\text{.}\)

Pick a vertex \(x\text{.}\) We will construct a directed tree \(T\) on \(V\) starting from \(x\) where every arc points away from \(x\text{.}\) This tree contains a directed path from \(x\) to any other vertex in the digraph, including our desired vertex \(y\text{.}\) We construct this directed tree as follows.

  • To start off, our tree \(T\) just contains the root vertex \(x\text{.}\)

  • Take \(A = \{ x \}\) and \(B = V \backslash \{ x \}\text{.}\) There is an arc from \(A\) to \(B\text{,}\) which must be of the form \((x, v_1)\) for some \(v_1 \in B\text{.}\) Add this arc and vertex to the tree \(T\text{.}\)

  • Next, take \(A = \{ x, v_1 \}\) and \(B = V \backslash A\text{.}\) There is an arc from \(A\) to \(B\text{,}\) so add this arc and its new head vertex \(v_2\text{.}\)

  • Next, take \(A = \{ x, v_1, v_2 \}\) and \(B= V \backslash A\text{.}\) This adds a new vertex \(v_3\) to our directed tree.

  • Repeat this process until you have a directed tree containing all of the vertices of \(V\text{,}\) including the desired vertex \(y\text{.}\)

You can make the same argument to create a directed tree rooted at \(y\text{,}\) which identifies a directed path from \(y\) to \(x\text{.}\)

Challenge: The \((\Leftarrow)\) proof can be made more rigorous by using induction on the size of \(V\text{.}\) I encourage you to write such a proof!

7. Transitive Tournaments.

A tournament \(T\) is transitive if whenever \((u,v)\) and \((v,w)\) are arcs in \(T\text{,}\) then \((u,w)\) is also an arc of \(T\text{.}\)

  1. Draw all possible unlabeled transitive tournaments on 3 vertices and 4 vertices.

  2. Prove that a tournament \(T\) is transitive if and only if \(T\) has no (directed) cycles. (Hint: prove \(\Longrightarrow\) by contradiction by picking a cycle of shortest length, then showing there is an even shorter cycle!)

Solution.

  1. In fact, there is only one way to make a transitive tournament on \(n\) unlabelled vertices. In particular, there will always be a source vertex with all arcs pointing outwards, and a sink vertex with all arcs pointing inwards.

  2. \((\Rightarrow)\) Suppose that \(T\) is a transitive tournament that contains a cycle. Pick a cycle \(C\) of shortest length on vertices \(v_1, v_2, \ldots, v_k\text{.}\) Note that \(k >3\) since the definition of a transitive tournament forbids a directed 3-cycle. Look at the arc between \(v_1\) and \(v_3\text{.}\) Since \(v_1 v_2\) and \(v_2 v_3\) are arcs, this forces \(v_1 v_3\) to be an arc. But then \(v_1, v_3, \ldots v_k\) is a directed cycle whose length is shorter than \(C\text{,}\) a contradiction. Therefore \(T\) cannot contain any directed cycles.

    \((\Leftarrow)\) Suppose that \(T\) has no directed cycles. Suppose that \((u,v)\) and \((v,w)\) are arcs. We cannot have a directed 3-cycle, so \(u\) and \(w\) must be connected by the arc \((u,w)\text{.}\) Therefore, \(T\) is transitive.
