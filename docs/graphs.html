Chapter 1 Graphs and Digraphs

Exercises Practice Problems

1. Handshaking Lemma.

Let \(G=(V,E)\) be an undirected graph. Prove the "handshaking lemma," which states that

\begin{equation*} \displaystyle{2|E| = \sum_{v \in V} \deg(v)}. \end{equation*}
2. Digraph Handshaking Lemma.

Let \(D=(V,E)\) be a digraph. Prove that

\begin{equation*} |E| = \sum_{v \in V} \deg^-(v) = \sum_{v \in V} \deg^+(v). \end{equation*}
3. Counting Graphs.

Consider the vertex set \(V=[n] = \{ 1, 2, 3, \ldots, n \}\text{.}\) We want to construct a simple, undirected graph on \(V\text{.}\)

  1. How many possible edges are there between these vertices?

  2. How many different simple graphs can we construct using these vertices? (The graphs do not have to be connected.)

4. Counting Digraphs.

Answer the same questions as in Problem 3, but this time for a simple directed graph.

5. Strongly Connected Components.

Find the strongly connected components of the following digraphs.

6. Strongly Connected Digraph.

Prove that a digraph \(D=(V,E)\) is strongly connected if and only if for every partition \(V = A \cup B\) into disjoint, nonempty sets, there is an arc in \(D\) directed from \(A\) to \(B\) and an arc in \(D\) directed from \(B\) to \(A\text{.}\)

7. Transitive Tournaments.

A tournament \(T\) is transitive if whenever \((u,v)\) and \((v,w)\) are arcs in \(T\text{,}\) then \((u,w)\) is also an arc of \(T\text{.}\)

  1. Draw all possible unlabeled transitive tournaments on 3 vertices and 4 vertices.

  2. Prove that a tournament \(T\) is transitive if and only if \(T\) has no (directed) cycles. (Hint: prove \(\Longrightarrow\) by contradiction by picking a cycle of shortest length, then showing there is an even shorter cycle!)
