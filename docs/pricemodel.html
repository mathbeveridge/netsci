Chapter 12 Price's Model for a Citation Network

Here is Price's Model for a citation network. Start with a small set of initial papers (with no citations). When a new paper is added, it cites \(c\) existing papers (on average). Each paper is chosen as follows:

  • with \(\displaystyle{\P = \frac{a}{a+c}}\text{,}\) cite a uniformly random paper

  • with \(\displaystyle{\P = \frac{c}{a+c}}\text{,}\) cite a paper with probability proportional to its current citation count.

The tail of the degree distribution satisfies the power law

\begin{equation*} \P(\deg(v) = k) \sim k^{-2-a/c}. \end{equation*}

Exercises Practice Problems

1. Link probabilities in Price's Model.

Suppose that we have run Price's Model and that the network has \(n\) vertices. We now add vertex \(n+1\text{,}\) as well as \(c\) arcs from vertex \(n+1\) to the existing network. Let \(k_i\) denote the degree of vertex \(i\text{.}\) Prove that for each new arc we add from vertex \(n+1\text{,}\)

\begin{equation*} \P(\mbox{the arc points to vertex } i) = \frac{a+k_i}{n(c+a)}. \end{equation*}
2. Price's Model via edge copying.

One critique of Price's Model is that the new paper must know about the degrees of the entire network when we choose its citations. So this is a "global" process instead of a "local" process. Consider the following process instead:

  • with \(\displaystyle{\P = \frac{a}{a+c}}\text{,}\) pick a paper uniformly at random. Cite this paper.

  • with \(\displaystyle{\P = \frac{c}{a+c}}\text{,}\) pick a directed edge uniformly at random. Then cite the paper that the edge points to.

Show that when we add vertex \(n+1\text{,}\) this local process ALSO satisfies

\begin{equation*} \P(\mbox{the arc points to vertex } i) = \frac{a+k_i}{n(c+a)}. \end{equation*}
3. The Vertex Copying Model.

Here is a different citation generation model. Start with a network of \(n_0 > c\) papers, each pointing randomly to \(c\) others. Pick a constant \(0 < \gamma < 1\text{.}\) When paper \(n+1\) is added, it makes \(c\) citations as follows:

  • Pick a random paper \(v\) from the existing \(n\) vertices.

  • For each of the \(c\) citations of \(v\) (that is, the arcs starting at \(v\)),

    • with \(\displaystyle{\P = \gamma}\text{,}\) copy that citation

    • with \(\displaystyle{\P = 1 - \gamma}\text{,}\) cite a uniform random paper instead.

You will now show that this model is equivalent to Price's model, so it generates a power law degree distribution.

  1. Show that for each arc starting at vertex \(n+1\text{,}\) we have

    \begin{equation*} \P(\mbox{the arc points to vertex } i) = (1- \gamma) \frac{1}{n} + \gamma \frac{k_i}{cn}. \end{equation*}

  2. Define \(a = c \left( \frac{1}{\gamma} -1 \right)\text{.}\) Confirm that this equation is equivalent to \(\gamma = \frac{c}{c+a}\text{.}\)

  3. Plug \(\gamma = \frac{c}{c+a}\) into equation from part (a) and simplify to show that we once again get

    \begin{equation*} \P(\mbox{the arc points to vertex } i) = \frac{a+k_i}{n(c+a)}. \end{equation*}

  4. Finally, rewrite the power law

    \begin{equation*} \P(\deg(v) = k) \sim k^{-2-a/c} \end{equation*}

    in terms of \(\gamma\) instead of \(a\) and \(c\text{.}\)
