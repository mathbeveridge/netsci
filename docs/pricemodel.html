Section 4.4 Price's Model for a Citation Network

Here is Price's Model for a citation network. Start with a small set of initial papers (with no citations). When a new paper is added, it cites \(c\) existing papers (on average). Each paper is chosen as follows:

  • with \(\displaystyle{\P = \frac{a}{a+c}}\text{,}\) cite a uniformly random paper

  • with \(\displaystyle{\P = \frac{c}{a+c}}\text{,}\) cite a paper with probability proportional to its current citation count.

The tail of the degree distribution satisfies the power law

\begin{equation*} \P(\deg(v) = k) \sim k^{-2-a/c}. \end{equation*}

Exercises Practice Problems

1. Link probabilities in Price's Model.

Suppose that we have run Price's Model and that the network has \(n\) vertices. We now add vertex \(n+1\text{,}\) as well as \(c\) arcs from vertex \(n+1\) to the existing network. Let \(k_i\) denote the degree of vertex \(i\text{.}\) Prove that for each new arc we add from vertex \(n+1\text{,}\)

\begin{equation*} \P(\mbox{the arc points to vertex } i) = \frac{a+k_i}{n(c+a)}. \end{equation*}

Solution.

The probability an the arc points to vertex \(i\) is

\begin{align*} \frac{a}{a+c} \cdot \frac{1}{n} + \frac{c}{a+c} \cdot \frac{k_i}{\sum_j k_j} &= \frac{a}{n(a+c)} + \frac{c}{a+c} \cdot \frac{k_i}{n c}\\ &= \frac{a+k_i}{n(a+c)} \end{align*}

2. Price's Model via edge copying.

Consider the following network growth model.

  • with \(\displaystyle{\P = \frac{a}{a+c}}\text{,}\) pick a paper uniformly at random. Cite this paper.

  • with \(\displaystyle{\P = \frac{c}{a+c}}\text{,}\) pick a directed edge uniformly at random. Then cite the paper that the edge points to.

Show that when we add vertex \(n+1\text{,}\) this local process ALSO satisfies

\begin{equation*} \P(\mbox{the arc points to vertex } i) = \frac{a+k_i}{n(c+a)}. \end{equation*}

Solution.

With probability \(\frac{a}{a+c}\text{,}\) we pick a random vertex. With probability \(\frac{c}{a+c}\text{,}\) we pick a random arc, and then take its target vertex. Among the \(cn\) arcs, \(k_i\) of them end at vertex \(i\text{.}\) This gives the same equation

\begin{equation*} \frac{a}{a+c} \cdot \frac{1}{n} + \frac{c}{a+c} \cdot \frac{k_i}{cn} \end{equation*}

as in the previous question.

3. The Vertex Copying Model.

One critique of Price's Model is that the new paper must know about the degrees of the entire network when we choose its citations. So this is a "global" process instead of a "local" process. Consider the following process instead. Start with a network of \(n_0 > c\) papers, each pointing randomly to \(c\) others. Pick a constant \(0 < \gamma < 1\text{.}\) When paper \(n+1\) is added, it makes \(c\) citations as follows:

  • Pick a random paper \(v\) from the existing \(n\) vertices.

  • For each of the \(c\) citations of \(v\) (that is, the arcs starting at \(v\)),

    • with \(\displaystyle{\P = \gamma}\text{,}\) copy that citation

    • with \(\displaystyle{\P = 1 - \gamma}\text{,}\) cite a uniform random paper instead.

You will now show that this model also generates a power law degree distribution.

  1. Show that for each arc starting at vertex \(n+1\text{,}\) we have

    \begin{equation*} \P(\mbox{the arc points to vertex } i) = (1- \gamma) \frac{1}{n} + \gamma \frac{k_i}{cn}. \end{equation*}

    Solution.

    • The probability that we consider copying a citation to paper \(i\) is \(\frac{k_i}{cn}\text{.}\) The probability that we do copy that citation is \(\gamma\text{.}\) We try this \(c\) times.

    • The expected number of random citations we make is \(c(1-\gamma)\text{.}\) A paper has probability \(\frac{1}{n}\) of being chosen each time.

    Note: Assuming that \(n\) is large compared to \(c\text{,}\) the probability that \(i\) gets multiple citations is negligible, so below, we will use the approximation \(\P\)(paper \(i\) receives a citation) \(\approx \E(\#\) citations received by paper \(i\)).

  2. Define \(a = c \left( \frac{1}{\gamma} -1 \right)\text{.}\) Confirm that this equation is equivalent to \(\gamma = \frac{c}{c+a}\text{.}\) Solution.

    \begin{align*} \frac{a}{c} &= \frac{1}{\gamma} -1\\ \frac{c+a}{c} &= \frac{1}{\gamma}\\ \frac{c}{c+a} &= {\gamma} \end{align*}

  3. Plug \(\gamma = \frac{c}{c+a}\) into equation from part (a) and simplify to show that we once again get

    \begin{equation*} \P(\mbox{the arc points to vertex } i) = \frac{a+k_i}{n(c+a)}. \end{equation*}

    Solution.

    \begin{align*} \P(\mbox{the arc points to vertex } i) &= (1- \gamma) \frac{1}{n} + \gamma \frac{k_i}{cn}\\ &= \left( 1 - \frac{c}{c+a} \right) \frac{1}{n} + \frac{c}{c+a} \frac{k_i}{cn}\\ &= \frac{a}{c+a} \frac{1}{n} + \frac{c}{c+a} \frac{k_i}{cn}\\ &= \frac{a+k_i}{n(c+a)} \end{align*}

  4. Finally, rewrite the power law

    \begin{equation*} \P(\deg(v) = k) \sim k^{-2-a/c} \end{equation*}

    in terms of \(\gamma\) instead of \(a\) and \(c\text{.}\) Solution.

    The degree distribution is governed by the power law

    \begin{equation*} p_k \sim k^{-2-a/c} = k^{-2- 1/\gamma +1} = k^{-1 - 1/\gamma} \end{equation*}

    where \(0 < \gamma < 1\text{.}\) Depending on the value of \(\gamma\text{,}\) our power law exponent \(\alpha\) can take on any value \(\alpha > 2\text{.}\)

  5. This vertex copying model leads to the same equations for the degree distribution of the resulting network. However, the actual network structure of the vertex copying model is very different from the netwok structure of Price's Model. Explain why. (Hint: start by thinking about what the vertex copying model would look like when \(\gamma\) is very very close to 1.) Solution.

    When \(\gamma\) is very very close to 1, each new vertex is practically a duplicate of the vertex that it is copying. So there is a strong correlation between the out-edges of the graph. This would lead to some community structure. However, this doesn't happens for Price's model, since the links are all taken uniformly at random from across the entire network. This out-edge correlation holds for all values of \(\gamma\text{,}\) but is more pronounced for larger \(\gamma\text{.}\)

4. Preferential Attachment Model.

Here is the Barabasi-Albert Preferential Attachmnent Model for an undirected network. We start with a complete graph on \(c+1\) vertices.

When we add the new vertex \(n+1\) to the network of size \(n\text{,}\) we create \(c\) undirected edges incident with \(n+1\text{.}\) The other endpoint \(i\) of each edge is chosen with probability proportional to its current degree \(k_i\text{.}\)

  1. Explain why the probability that an edge from vertex \(n+1\) attaches to vertex \(i\) is

    \begin{equation*} \frac{k_i}{2cn}. \end{equation*}

    Solution.

    The total number of edges in the network of order \(n\) is \(2cn\text{.}\) So the probability that we pick vertex \(i\) is

    \begin{equation*} \frac{k_i}{2cn}. \end{equation*}

  2. Now change this network model to a directed network. Show that

    \begin{equation*} k_i= k_i^{\mathrm{in}} + k_i^{\mathrm{out}} = k_i^{\mathrm{in}} + c. \end{equation*}

    Solution.

    When we add vertex \(i\text{,}\) its out-degree is \(c\) and its in-degree is 0. Later on, if its total degree is \(k_i\text{,}\) then we still have \(k_i^{\mathrm{out}}=c\text{,}\) and so \(k_i^{\mathrm{in}} = k_i - c,\text{.}\)

  3. Now show that Preferential Attachment is just the undirected version of Price's Model with \(a=c\text{.}\) Solution.

    The probability that an edge from vertex \(n+1\) is incident with vertex \(i\) is

    \begin{equation*} \frac{k_i}{2cn} = \frac{c + k_i^{\mathrm{in}}}{2cn} = \frac{1}{2} \frac{1}{n} + \frac{1}{2} \frac{k_i^{\mathrm{in}}}{cn} \end{equation*}

    and this is Price's model with \(a=c\text{.}\)
