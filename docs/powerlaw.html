Chapter 11 Power Laws

Exercises Practice Problems

1. Degree Distribution.

The degree distribution of a graph is given by

\begin{equation*} p_k = \mbox{ the fraction of vertices with degree } k. \end{equation*}

What is the degree distribution \((p_1, p_2, \ldots)\) of this graph?

Solution.

\begin{equation*} \left( 0,\frac{4}{10},\frac{2}{10},\frac{3}{10},0,\frac{1}{10} \right) \end{equation*}

2. The Normalizing Constant.

We are given a data set whose degree distribution looks like a straight line with negative slope when plotted using a log-log scale. In other words,

\begin{equation*} \ln (p(k)) = c - \alpha \ln (k), \end{equation*}

where \(c >0\) and \(\alpha > 0\) are constants. Exponentiate both sides to show that

\begin{equation*} p(k) = C k^{-\alpha}. \end{equation*}

What is the value of the constant \(C\) in terms of \(c\text{?}\) Solution.

\begin{equation*} p(k) = e^c \, k^{-\alpha} \qquad \mbox{so } C = e^c \end{equation*}

3. The Normalizing Constant for Exponent \(\alpha\).

  1. Let \(p: \Z^+ \rightarrow [0,1]\) given by

    \begin{equation*} p(k) = C k^{- \alpha} \end{equation*}
    be the PMF for a power law. (Here, we assume that \(p(0)=0\) for convenience.) Suppose that we know the value for \(\alpha\text{.}\) Prove that
    \begin{equation*} C = \frac{1}{\zeta(\alpha)} \end{equation*}
    where \(\zeta(x)\) is the Riemann zeta function
    \begin{equation*} \displaystyle{\zeta(x) = \sum_{k=1}^{\infty} k^{-x}}. \end{equation*}
    What is the restriction on allowable values of \(\alpha\text{?}\) Solution.

    We need \(\displaystyle{\zeta(x) = \sum_{k=1}^{\infty} k^{-x}}\) to converge so that the constant \(C\) is finite. From calculus, we recall that the series

    \begin{equation*} \sum_{n=1}^{\infty} n^{-r} = \sum_{n=1}^{\infty} \frac{1}{n^r} \end{equation*}

    converges if and only if \(r > 1\text{.}\) Translating this to our power law, we set \(k=n\) and \(\alpha=r\text{.}\) We must have \(\alpha > 1\) in order for the constant \(C\) to be finite.

  2. For many real-world networks, the power law behavior doesn't hold for small \(k\text{.}\) In this case, we restrict our attention to vertices of degree at least \(k_{\min}\text{.}\) We then deal with the probability mass function for a truncated power law. Given \(\alpha\) and \(k \geq k_{\min}\) this PMF is given by

    \begin{equation*} p(k) = \frac{ k^{-\alpha}}{\zeta(\alpha, k_{\min})} \end{equation*}

    where

    \begin{equation*} \zeta(x, k_{\min}) = \sum_{k=k_{\min}}^{\infty} k^{-x} \end{equation*}

    is the generalized Riemann zeta function.

    Dealing with the Riemann zeta function is unwieldy. In practice, we approximate this leading constant by the corresponding integral. Evaluate the following integral in order to find the approximate value for \(\zeta(\alpha, k_{\min})\text{:}\)

    \begin{equation*} A = \int_{x= k_{\min}}^{\infty} x^{-\alpha} \, dx \approx \sum_{k= k_{\min}}^{\infty} k^{-\alpha} . \end{equation*}

    Solution.

    \begin{align*} \sum_{k= k_{\min}}^{\infty} k^{-\alpha} &\approx \int_{x= k_{\min}}^{\infty} x^{-\alpha} \, dx\\ &= \lim_{n \rightarrow \infty} \frac{1}{-\alpha+1} x^{- \alpha+1} \Big|_{x=k_{\min}}^n\\ & = \lim_{n \rightarrow \infty} \frac{1}{-\alpha+1} \left( \frac{1}{n^{\alpha-1}} - \frac{1}{k_{\min}^{\alpha-1}}\right)\\ &= \frac{1}{\alpha-1} \cdot \frac{1}{k_{\min}^{\alpha-1}} \end{align*}

    provided that \(\alpha > 1\text{,}\) of course!

  3. Find the corresponding approximate expression for the power law PMF

    \begin{equation*} p(k) = \frac{1}{A} k^{-\alpha} \end{equation*}

    using your constant \(A\) from (a). Simplify so that \(\alpha\) only appears once in an exponent. Solution.

    \begin{equation*} p(k) \approx \left( \frac{1}{\alpha-1} \cdot \frac{1}{k_{\min}^{\alpha-1}} \right)^{-1} k^{- \alpha} = \frac{\alpha -1}{k_{\min} } \left( \frac{ k_{\min}}{k} \right)^{\alpha} \end{equation*}

4. Approximating with a Continuous Power Law.

In general, it is easier to pretend that we are dealing with a continuous distribution (rather than a discrete distribution). So we will talk about a probability density function (PDF) \(p: [k_{\min}, n] \rightarrow [0,1]\) given by

\begin{equation*} p(x) = C x^{-\alpha} \end{equation*}

rather than a PMF. You can find the constant \(C = (\alpha-1)k_{\min}^{\alpha-1} \) as you did in the previous problem. But let's just keep calling it \(C\) for now.

  1. Let \(X\) be a random variable governed by the continuous PDF \(p(x)\text{.}\) Find

    \begin{equation*} \E[X] = \int_{x=k_{\min}}^{n} x p(x) dx. \end{equation*}

    Solution.

    \begin{align*} \E[X] &= C \int_{x=k_{\min}}^{n} x \cdot x^{-\alpha} dx = C \int_{x=k_{\min}}^{n} x^{-\alpha+1} dx\\ &= \frac{C}{-\alpha +2} x^{-\alpha+2} \Big|_{k_{\min}}^n\\ &= \frac{C}{-\alpha +2} \left( \frac{1}{n^{\alpha-2}} - \frac{1}{k_{\min}^{\alpha-2}} \right) \end{align*}

  2. The \(m\)th moment of \(X\) is

    \begin{equation*} \E[X^m] = \int_{x=k_{\min}}^{n} x^m p(x) dx. \end{equation*}

    Calculate it. Solution.

    \begin{align*} \E[X^m] &= C \int_{x=k_{\min}}^{n} x^m \cdot x^{-\alpha} dx = C \int_{x=k_{\min}}^{n} x^{-\alpha+m} dx\\ &= \frac{C}{-\alpha +m+1} x^{-\alpha+n+1} \Big|_{k_{\min}}^n\\ &= \frac{C}{-\alpha +m+1} \left( \frac{1}{n^{\alpha-m-1}} - \frac{1}{k_{\min}^{\alpha-m-1}} \right) \end{align*}

  3. Look at your answers for parts (a) and (b) and think about what happens as \(n \rightarrow \infty\text{.}\) For what values of \(\alpha\) does \(\E[X]\) converge? When does the \(m\)th moment \(\E[X^m]\) converge? Solution.

    • As \(n \rightarrow \infty\text{,}\) the expected value \(\E[X]\) is

      \begin{equation*} \lim_{n \rightarrow \infty} \frac{C}{-\alpha +2} \left( \frac{1}{n^{\alpha-2}} - \frac{1}{k_{\min}^{\alpha-2}} \right) = \frac{C}{\alpha-2} \cdot \frac{1}{k_{\min}^{\alpha-2}} \end{equation*}

      provided that \(\alpha > 2\text{.}\) So the average value is finite if and only if \(\alpha > 2\text{.}\) Finally, let's substitute in our approximation of the constant \(C = (\alpha-1)k_{\min}^{\alpha-1}\) to find that the expected value is

      \begin{equation*} \E[X] = \frac{\alpha-1}{\alpha-2} \cdot \frac{k_{\min}^{\alpha-1}}{k_{\min}^{\alpha-2}} = \frac{\alpha-1}{\alpha-2} \cdot k_{\min}. \end{equation*}

      That's pretty cool. And you can see that if your exponent \(\alpha \rightarrow 2\) then the expected value goes to infinity. Nice!

    • As \(n \rightarrow \infty\text{,}\) the \(m\)th moment \(\E[X^m]\) is

      \begin{equation*} \lim_{n \rightarrow \infty} \frac{C}{-\alpha +m+1} \left( \frac{1}{n^{\alpha-m-1}} - \frac{1}{k_{\min}^{\alpha-m-1}} \right) = \frac{C}{\alpha-m-1} \cdot \frac{1}{k_{\min}^{\alpha-m-1}} \end{equation*}

      provided that \(\alpha > m\text{.}\) So the \(m\)th moment value is finite if and only if \(\alpha > m\text{.}\) Let's substitute \(C= (\alpha-1)k_{\min}^{\alpha-1}\) to get the expression

      \begin{equation*} \E[X^m] = \frac{\alpha-1}{\alpha-m-1} \cdot \frac{k_{\min}^{\alpha-1}}{k_{\min}^{\alpha-m-1}} = \frac{\alpha-1}{\alpha-m-1} \cdot k_{\min}^{m}. \end{equation*}

  4. Recall that most observed power laws have \(2 \leq \alpha \leq 3\text{.}\) In this range, does the expected value converge? Which moments converge, and which moments diverge? Solution.

    So why do most naturally occurring power laws have \(2 < \alpha < 3\text{?}\) We'll consider the degree sequence for a network. In this range of \(\alpha\text{,}\) the average degree is finite. This makes sense for our graph. Next, we observe that the second moment is infinite (and so are the higher moments). This means that the variance of the degrees goes to infinity. Therefore, we expect to find hub nodes with very, very high degree. If \(\alpha > 3\text{,}\) then the second moment would converge, which means that the degrees would be well-concentrated around the expected value. We've already seen how important the hub nodes are in the network. So if hub nodes only occur when \(\alpha < 3\text{,}\) it makes sense that our observed power laws should fall into the range of \(2 < \alpha < 3\text{.}\)

    What about other natural power laws, like the size of cities? Remember that a power law is the correct model for a "popularity contest" or a "rich get richer phenomenon." In these instances, there are some incredibly successful entities, and their success breeds even more success. In other words, in such popularity feedback loops, we anticipate seeing entities that are vastly more successful than the average entity. This can only happen when the variance of the power law is infinite.
